# Git tags
Tutorial d'etiquetes.

## Etiquetes lleugeres
Una etiqueta lleugera sols és una referència a un commit existent.

## Etiquetes anotades
Una etiqueta anotada té. a més de la referència, la persona que ha creat l'etiqueta i la data de creació.
